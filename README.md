# CRUD by Phalcon

 CRUD (Create, Read, Update and Delete) using Phalcon Framework

## Tools dan Framework
* Apache 2.4.25
* MariaDB 10.1.23
* PHP 7.1.3
* Phalcon 3.2
* Boostrap 3.3.7

## Getting started

Create database with name 'crud_by_phalcon' and execute file .sql for create table and dummy data.

## Run 

(http://localhost/crud_by_phalcon/)
